## App Features: 

- Horizontally scroll between the jokes
- Add new Joke.
- Gradient Background.
- 3D Rotation Effect for each Joke Card.
- Store the Jokes using CoreData.
- Remove Jokes by swipe the Joke Card up.
- Supports Dark Mode.
